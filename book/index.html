The book

Dimensional Scripture

When Physics Meets Prophecy.

Dimensional Scripture is a research-driven manuscript-in-progress examining consciousness, layered reality, and the biblical text. It is not devotional literature and it is not spiritual marketing.

The project asks a narrow but difficult question: what if ancient authors were describing real structures of reality using the language available to them, and what if modern materialism has simply failed to account for those structures?
